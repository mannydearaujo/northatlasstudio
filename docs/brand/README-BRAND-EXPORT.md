# NorthPoint Digital — Brand Export Pack

This folder contains source-ready brand assets for the approved NorthPoint Digital visual direction.

## Important
- `index.html` was **not edited**.
- These files are staged for export/import into the repo source folder.
- Recommended repo destination: `assets/brand/`, `styles/`, and `docs/brand/`.

## Approved direction
Modern professional tech brand with:
- Deep navy / carbon background
- Neon cyan accent
- Filled north-arrow / compass mark
- White Inter-style wordmark
- Subtle grid + data-flow motifs
- Local search, lead tracking, analytics, and AI discovery visual language

## Core assets
- `assets/brand/svg/northpoint-logo-primary.svg`
- `assets/brand/svg/northpoint-logo-stacked.svg`
- `assets/brand/svg/northpoint-mark.svg`
- `assets/brand/svg/favicon.svg`

- `styles/northpoint-brand.css` contains reusable CSS tokens/classes.

## Implementation status — done
The mark is live in production: the nav logo, favicon, and apple-touch-icon all reference
this pack (`assets/brand/svg/northpoint-mark.svg`, `assets/brand/svg/favicon.svg`,
`assets/brand/png/favicon-512.png`). Verified rendering correctly on
`https://northpoint-digital.vercel.app` as of 2026-06-19. This pack remains the
brand source-of-truth for anything not yet on the site (social banners, proposal
covers, future hero treatments).
