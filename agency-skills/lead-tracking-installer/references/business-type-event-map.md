# Business Type Event Map

Phone-driven: real `phone_tap`, secondary `sms_tap`, `email_click`. Quote-driven: real `quote_request_submit`, `phone_tap`; secondary `quote_request_click`. Booking-driven: real `booking_click`, `phone_tap`; secondary `booking_embed_interaction`; diagnostic `booking_embed_load`. Ecommerce: use GA4 recommended ecommerce events, real `purchase`.
