# Event Naming Conventions

Use lowercase snake_case. Real leads: `phone_tap`, `quote_request_submit`, `booking_click`, `contact_form_submit`, `appointment_request_submit`, `purchase`. Secondary: `sms_tap`, `email_click`, `quote_request_click`, `form_start`, `booking_embed_interaction`. Diagnostics: `booking_embed_load`, `form_view`, `component_visible`.
