# Prioritization Rubric

Score each candidate 0–2 per dimension. Sort by total. Business relevance = 0 kills the topic outright.

| Dimension | 0 | 1 | 2 |
|---|---|---|---|
| **Business relevance** | No path to a lead | Supports awareness for a real service | Directly answers a pre-hire question for a core service |
| **Local specificity** | Could be about anywhere | Area-relevant angle | Names verifiable local conditions/rules/places (owner-confirmed) |
| **Proof availability** | Nothing to show | Some owner knowledge | Real photos, real jobs, real numbers available |
| **Internal-link value** | Links to nothing | Links to 1 money page | Natural hub linking 3+ service/location pages |
| **Timing** | Evergreen, no urgency | Evergreen + useful now | Seasonal window approaching (schedule it) |

**8–10:** brief it now. **5–7:** backlog, pull for calendar gaps. **≤4:** drop.

Tie-breaker: which topic does the owner get asked about most on the phone? Phone-question frequency is ground-truth search intent for local service.
