# Review Request Templates

Adapt to the client's voice pack. The ask is always honest-review, never positive-review.

## SMS (best conversion for local service — send same/next day)
> Hi [first name], it's [owner] from [business]. Thanks for having us out today for [the job, specific]. If you have 60 seconds, an honest Google review helps other [town] folks find us: [short review link]

## Email
Subject: Thanks from [business] — one small favor

> Hi [name],
> Thanks for trusting us with [specific job] this week. If you have a minute, would you leave an honest review of how it went? It's the main way neighbors in [town] find us.
> [Review button/link]
> If anything wasn't right, just reply to this email — I'd rather fix it than not know. — [owner]

("Reply if something wasn't right" is an open door, NOT a filter — the review link goes to everyone in the same message.)

## In-person + follow-up combo
Script for staff: "If you're happy with how [pet/job] turned out, a Google review really helps us — I can text you the link right now."
Then the SMS above within the hour.

## The one reminder (5–7 days, then stop)
> Hi [name], [owner] here — no pressure at all, just resending the review link in case it got buried: [link]. Thanks again for the business either way!

## Setup notes
- Get the canonical short link from GBP admin (Share profile → review link); test it logged out.
- Log asks (date, customer, channel) so velocity is measurable and nobody gets double-asked.
