# Voice Pack File Templates

## tone.md
```markdown
# [Client] — Tone
> Read all five voice files before writing ANY copy for this client.

Register: [casual-professional / warm-premium / blunt-tradesman …]
Person: [we/I], contractions [yes/no]
Rhythm: [e.g., short sentences, average 12 words; one-line paragraphs common]
Price talk: [exact framing rules, e.g., "cash discount" never "card price"; never defensive about premium pricing]

Calibration lines (owner verbatim — match this energy):
1. "[exact owner quote]"
2. "[exact owner quote]"
3. "[exact owner quote]"
```

## banned-phrases.md
```markdown
# [Client] — Banned
Client-specific: [e.g., "one dog at a time", em-dashes anywhere on site]
Unsupported claims: [e.g., "best in Boston", "guaranteed lowest price"]
Universal AI-tells: inherit the full list from the human-copy-editor skill; do not restate.
```

