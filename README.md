# Manny de Araujo — Local Growth Agency

Conversion-focused landing page for a growth agency serving local service businesses:
websites, local SEO, and lead tracking.

Static single-file site (`index.html`). Deployed on Vercel.

## Develop
Just open `index.html` in a browser — no build step.

## Deploy
Pushes to `main` auto-deploy via the Vercel Git integration.
