# Reports Folder

Use this folder for deliverables and owner-facing reports.

Default polished deliverables should use NorthPoint Digital's KPI dashboard HTML
style from the branding package: charcoal grid background, cyan north-arrow
mark, compact KPI cards, and plain-English panels for results, strengths,
weaknesses, corrections needed, and next steps. Use PDF exports only when a
client specifically needs an attachment or print-ready copy.

Examples:

- Free Site Audit report
- Paid AI Search Readiness Audit
- Weekly client reports
- Launch QA summary
- Proposal or offer sheet exports
- Case study drafts
