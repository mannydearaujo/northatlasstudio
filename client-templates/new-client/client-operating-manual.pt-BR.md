- `phone_tap`

Regra de configuração:

- Pedir primeiro acesso ao GA4, GTM, Search Console, site/CMS e domínio/DNS.
- Se o acesso antigo estiver perdido, tentar recuperar ou verificar propriedade.
- Se GA4/GTM não puderem ser recuperados, criar uma nova propriedade GA4 e/ou novo container GTM e registrar a data de instalação.
- Não afirmar performance histórica de GA4/Search Console se o acesso ou os dados não existirem.
- Marcar apenas leads reais como Key Events no GA4.
- Usar Microsoft Clarity, CallRail/WhatConverts, Formspree/HubSpot, Looker Studio e ferramentas externas como complementos quando fizer sentido.
