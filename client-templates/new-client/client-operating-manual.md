## Tracking Rules

Real-lead events:

- `phone_tap`
- `[quote_request_submit / booking_click / contact_form_submit]`

Secondary events:

- `sms_tap`
- `email_click`
- `[quote_request_click / form_start / booking_embed_interaction]`

Diagnostic events:

- `[booking_embed_load / form_view / component_visible]`

Tracking setup rule:

- Ask for GA4, GTM, Search Console, website/CMS, and domain/DNS access first.
- If old access is missing, recover or verify ownership where possible.
- If GA4/GTM cannot be recovered, create a new GA4 property and/or GTM container and record the install date.
- Do not claim historical GA4/Search Console performance if access or data was unavailable.
- Mark only real-lead events as GA4 Key Events.
- Use Microsoft Clarity, CallRail/WhatConverts, Formspree/HubSpot forms, Looker Studio, and external SEO tools as add-ons when useful.
