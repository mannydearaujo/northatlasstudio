What Gets Tracked

Separate real leads from useful diagnostics.

A clean report does not inflate progress. Real leads are high-intent actions. Secondary intent explains behavior. Diagnostic events help confirm the website is working, but they are not reported as wins.

GTM, when used, is the tag control panel. GA4 is where the data gets reported. North Atlas explains both plainly so the owner knows what is installed and why.

FAQ

Tracking questions

What counts as a lead?

A real lead is an action with business intent: a phone tap, completed form, quote request, booking click, or appointment request.

Can you use existing GA4?

Yes, if the access and data are usable. If old access cannot be recovered, North Atlas can create a clean setup and report from the install date forward.

Do you report rankings?

Rankings and Search Console data can provide context, but the headline reporting focuses on leads and actions that point toward revenue.
