# North Atlas Studio Email Signature

Created: 2026-07-14

Files:

- `email-signature.html` - styled signature for Gmail or other HTML-capable email clients.
- `email-signature-with-banner.html` - banner-style signature option.
- `email-signature.txt` - plain-text fallback for cold outreach or clients that strip HTML.
- `email-signature-banner.svg` - small branded banner, 600x150, for places where a banner is useful.
- `email-signature-banner.png` - PNG export of the small banner, safer for email clients than SVG.

The HTML signature uses the public Atlas mark URL:

`https://www.northatlasstudio.com/assets/brand/atlas/atlas-mark-navy.svg`

If the SVG does not render in a specific email client, use the text-only signature or replace the image
with a hosted PNG profile/logo asset.

The small banner is intentionally separate from the default signature. Use it sparingly; cold outreach
usually performs better with the lighter text-and-mark signature.

Important: the banner HTML uses a local image path (`email-signature-banner.png`). For Gmail or another
email client, upload/insert the PNG directly or host it at a public URL and replace the image source.
